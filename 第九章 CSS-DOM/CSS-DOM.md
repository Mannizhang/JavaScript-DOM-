# 三位一体的网页
1. 结构层
- 网页的结构层由HTML或XHTML之类的标记语言负责创建。
2. 表示层
- 表示层由css负责完成。css描述页面内容应该如何呈现。
3. 行为层
- 行为层负责内容应该如何响应事件这一问题。
- 这是JavaScript语言和DOM主宰的领域。
4. 分离
- 三种技术之间存在着一些潜在的重叠区域。
